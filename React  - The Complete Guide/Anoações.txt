 - npm install create-react-app -g (-g Instalação global).
 - create-react-app 'nome_da_aplicacao' --script-version 1.1.5 (Versao utilizada no curso).
 - Para subir a aplicação, digitar npm start dentro da pasta da aplicacao ao digitar.
 - Prefira utilizar Classes quando você quiser utilizar estados, caso contrário utilize somente métodos.
 - Ao digitar "npm run eject", serão exibidos uma série de bibliotecas no package.json e exibidas as pastas de configuração e scripts.
 - ErroBoundary - somente utilize em trecho que vc sabe que algum erro pode acontecer.
 - npm install --save prop-types - Pode ser utilizado para tipar as variáveis dos componentes.
